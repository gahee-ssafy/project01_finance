#1
질문: 서울에 10억짜리 주택을 살려고 해 LTV는?

🤖 [사수 직답]:
**해당 주택은 취급 불가합니다. 담보주택 평가액이 6억원을 초과하는 주택은 보금자리론을 이용할 수 없습니다.**

매뉴얼 [제8장 담보주택] 1. 주택가격 기준에 따라, 구입용도의 경우 시세정 보, 감정평가액, 매매가액 중 어느 하나라도 6억원을 초과하는 주택은 취급이 불가합니다. 따라서 10억원 주택은 LTV 산정 대상에 해당하지 않습니다.

📊 퍼플렉시티 통계 (상위 매칭)
   - 주 근거: 제8장 담보주택 (유사도: 0.5136)

 #2
 질문: 수도권 외 지역에서 6억짜리 집을 살려고 해 전세보증금은 3억이다
LTV 한도는?
🤖 [사수 직답]:
비규제지역 아파트는 70%, 기타주택은 65%입니다. 규제지역은 아파트 60%, 기타주택 55%가 적용됩니다.

**상세 안내:**
1. **LTV 한도(비규제지역):** 아파트는 **70%**, 기타주택(연립, 다세대, 단독 등)은 **65%** 이내입니다.
2. **LTV 한도(규제지역):** 아파트는 **60%**, 기타주택은 **55%** 이내입니다.
3. **산정 방식:** LTV는 (대출금액 + 선순위채권 + **임대보증금** 및 최우 선변제 소액임차보증금) / 담보주택 평가액으로 계산합니다. 따라서 질문하신 전세보증금 3억 원은 LTV 산정 시 분자의 합산 항목에 포함됩니다.
4. **주택가격 기준:** 구입용도의 경우 시세, 감정가, 매매가 중 어느 하나 라도 6억 원을 초과하면 취급이 불가하나, 현재 6억 원이므로 요건을 충족합 니다.

📊 퍼플렉시티 통계 (상위 매칭)
   - 주 근거: 제3장 자금용도 (유사도: 0.6368)
   - 분석 범위: 상위 3개 챕터 통합 분석
